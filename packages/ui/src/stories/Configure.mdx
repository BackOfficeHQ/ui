    # Back Office Design System

    <a href="https://design.backoffice.co.th" target="_blank">Official Website</a>

    Back Office Design System is a comprehensive collection of reusable components, guidelines, and best practices aimed at creating a consistent and efficient user interface for back office applications. It ensures that all products share a unified look and feel, improving user experience and development speed.

    Key features include:

    - **Consistency**: Standardized components and styles across all applications.
    - **Efficiency**: Accelerated development with pre-built, tested components.
    - **Scalability**: Easily adaptable to new requirements and scalable for large projects.
    - **Accessibility**: Designed with accessibility in mind to ensure inclusivity.
